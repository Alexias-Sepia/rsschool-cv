## Пример кода:
Простая программа для вывода таблицы умножения с помощью С++ и библиотеки stdio.
```
#include <stdio.h>
int main(){
  int i, j;
  char c;
  for(i = 1; i <= 9 ; i++) {
    for(j = 1; j <= 9 ; j++) {
    printf ("%5d", i * j);}
    printf("\n");}
  c = getchar();
  return 0;
}
```
